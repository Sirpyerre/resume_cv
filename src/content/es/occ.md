# OCC Mundial — Backend Developer
**Julio 2022 — Julio 2025**

**Resumen**
Participé activamente en la migración e integración de una plataforma unificada. Me enfoqué en servicios backend, rendimiento, observabilidad y CI/CD para garantizar entregas confiables y escalables.

**Logros clave**
- Participé en la integración de una plataforma unificada escrita en C#, consolidando endpoints para consumo por mobile y web.
- Desarrollé y mantuve APIs REST asegurando la integración con servicios externos y mejorando la eficiencia del flujo de datos.
- Integré Elasticsearch para mejorar las capacidades de búsqueda, optimizando el indexado y el rendimiento de consultas.
- Desarrollo de API GraphQL: construí una capa GraphQL para agregar respuestas de múltiples APIs REST, simplificando el consumo desde frontend.
- Desarrollé una aplicación backend en Go que consumía mensajes desde Kafka, enviaba datos a APIs externas y confirmaba el mensaje solo cuando la API externa devolvía éxito (200), reduciendo consultas concurrentes a la base de datos y mejorando la confiabilidad.
- Contenericé aplicaciones con Docker para agilizar despliegues y gestión de recursos.
- Propuse e implementé mejoras de monitoring y observabilidad: dashboards en Grafana y exportadores/collectors para Prometheus para métricas y alertas en tiempo real.
- Automatización de pipelines CI/CD para construir y desplegar servicios alojados en los repositorios de la organización.
- Refactorización y mejora de calidad de código, aumentando cobertura de pruebas unitarias y usando linters para mantener estándares.
- Mantenimiento y soporte de código legado (Java, ASP.NET, C#) cuando fue necesario en tareas de migración.
