# BlackInTech — Developer Fullstack
**Marzo 2019 — Junio 2022**

**Resumen**
Trabajé en responsabilidades de backend y frontend, con foco en arquitecturas serverless, desarrollo de APIs e integración de servicios en AWS.

**Logros clave**
- Diseño y despliegue de funciones AWS Lambda con Go usando Serverless Framework para implementar features orientadas a eventos.
- Implementación de APIs con AWS API Gateway y Lambda, aplicando buenas prácticas de arquitecturas serverless y seguridad.
- Desarrollo de módulos backend en Symfony y creación de APIs con Laminas/Mezzio para distintos proyectos.
- Implementación de funcionalidades en React para consumo del frontend cuando el producto lo requería.
- Implementación de pipelines de CI/CD e integración continua con CircleCI.
- Desarrollo en contenedores y simulación local de servicios cloud cuando fue necesario para pruebas y desarrollo.
